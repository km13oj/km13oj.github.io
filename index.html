This is the user journey desigened as part of an OCAD prototype created with Laura Wright. HUSH solves the problem of noisy coworking and study spaces.

HUSH measures noise levels, and then displays that volume in three different ways. First, it tweets the average volume levels from @VolumeBot2013, so that the volume can be checked remotely.

Second, HUSH displays the current volume levels in the chosen space.

The projection data visualization is live, and we coded it using Processing.

Finally, HUSH displays historic volume trends online, so users can understand historic volume trends and choose a working space accordingly. HUSH connects two different spaces: the chosen space, and wherever the user is. In this case, the Digital Futures Initiative lab was chosen as a test space. A student can check how loud the space is before travelling from home to work there.
